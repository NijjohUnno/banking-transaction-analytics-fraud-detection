# Banking Transaction Analytics & Fraud Detection

## Project Overview

This project demonstrates an end-to-end Banking Transaction Analytics and Fraud Detection solution using **MySQL** for the database layer and **Power BI** for reporting and visualization.
The project simulates a real-world banking environment by generating synthetic customer, account, and transaction data, including intentional fraud scenarios such as:
- High-value transactions
- Velocity spikes
- Late-night transfers
- Unusual transaction behavior
- 
The solution focuses on:
- Advanced SQL analytics
- Fraud detection logic
- Database automation
- Query optimization
- Executive reporting
- Risk monitoring dashboards
